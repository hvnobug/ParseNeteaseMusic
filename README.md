# ParseNeteaseMusic
解析网易云音乐分享链接

#### 一个简单的 python 解析网易云音乐分享链接

name 歌曲名称

type 类型

lyric 歌词文本

artist 歌曲作者

album 专辑

duration  歌曲时长

description 歌曲描述

albumurl  专辑链接

artisturl 艺术家链接

lrcurl  歌词链接

songurl 歌曲链接

目前只支持单曲链接解析, 同时需要 [chromedriver](http://chromedriver.storage.googleapis.com/index.html) 支持
